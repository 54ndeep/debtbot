Setup Guide

Get Telegram alerts
on every form submit

3 steps, ~10 minutes. No third-party services needed.

🌐 Your Website Form
→
⚡ Notifier Server
→
✈️ Your Telegram
1
Create your Telegram Bot & get your Chat ID
Takes 2 minutes — done entirely in Telegram

A) Create the bot: Open Telegram and search for @BotFather. Send it /newbot, give your bot a name, and copy the Bot Token it gives you.

B) Get your Chat ID: Search for @userinfobot, send it any message, and it will reply with your Chat ID (a number like 123456789).

⚠️ Start a conversation with your new bot first — send it /start — otherwise it can't message you.
2
Deploy the Notifier Server
Free hosting on Railway (recommended) or any Node.js host

The server files are in your downloaded ZIP. Deploy them to Railway.app (free tier works fine):

1. Push the folder to a GitHub repo (or drag-drop to Railway).
2. In Railway → New Project → Deploy from GitHub Repo.
3. Go to Variables and add these two environment variables:

Environment Variables 
TELEGRAM_BOT_TOKEN = 123456:ABCdef...   # from BotFather
TELEGRAM_CHAT_ID   = 123456789           # from @userinfobot

4. Railway will give you a public URL like https://my-app.up.railway.app. Copy it — you'll need it in Step 3.

Alternative hosts: Render.com, Fly.io, Heroku, or any VPS. The server is plain Node.js with no database.
3
Add the snippet to your website
One <script> tag — works with any HTML form

Paste this snippet just before the closing </body> tag on any page that has a form. Replace YOUR_SERVER_URL with the Railway URL from Step 2.

website-snippet.html 
<!-- Paste before </body> -->
<script>
(function () {
  // 👇 Replace with your Railway URL
  const NOTIFY_URL = "https://YOUR_SERVER_URL/notify";

  document.addEventListener("submit", async function (e) {
    const form = e.target;
    const data = {};
    new FormData(form).forEach((v, k) => data[k] = v);

    try {
      await fetch(NOTIFY_URL, {
        method: "POST",
        headers: { "Content-Type": "application/json" },
        body: JSON.stringify(data),
      });
    } catch (err) {
      console.warn("Notifier:", err.message);
    }
  });
})();
</script>
